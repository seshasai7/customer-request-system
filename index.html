Master Notification Emails
These addresses get an instant CC on every new customer request, an instant notification every time a store manager changes a request's status, the weekly master change log, and the stale-pending alert when any request has been sitting in "Pending" for 7+ days with no action taken.
Weekly Digest Email (per store)

Each Sunday at 8:00 PM, every store gets an automatic email with all of the past week's pending requests — sent for free by Google Apps Script. You can also send the digest manually right now.

⚙️ Configure Auto-Send

Three scheduled jobs run from Apps Script: sendWeeklyDigest (Sundays 8pm — per-store digest), sendMasterChangeLog (Sundays 9pm — admin emails), and checkStalePending (every Monday 7am — admin emails). Change schedules in Apps Script → Triggers.
